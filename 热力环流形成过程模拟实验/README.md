# 《基于 AI 与 3D 交互可视化的热力环流实验过程模拟》

## 项目简介

本作品围绕中学地理“热力环流”知识点，利用 AI 辅助开发和 Three.js 3D 可视化技术，构建了一个可交互的地理实验模拟场景。学生可以通过切换实验阶段和调节地表温差，观察地表冷热不均如何引起空气垂直运动、近地面气压差异、水平气流以及完整闭合环流的形成过程。

## 课程任务对应说明

本项目属于地理实验媒体作品，面向中学地理课堂展示与学生探究学习。作品使用前端网页技术实现三维实验模拟，不使用后端、数据库、真实地图或任何需要密钥的 API，符合“基于 AI 结合 3D 可视化、交互式网页等技术开发地理实验过程模拟”的课程任务要求。

## 运行方式

项目文件夹：`热力环流形成过程模拟实验`

推荐运行方式：

解压后双击 `start-server.bat`，脚本会自动寻找一个空闲端口，启动本地静态服务，并用系统默认浏览器打开网页。

备用运行方式：

如果电脑没有安装 Python，也可以直接双击 `index.html` 打开。本版本已经改为普通脚本加载方式，不再使用 `type="module"` 或 import map，直接打开本地 HTML 文件也能加载 Three.js。

说明：本项目为纯前端静态网页。Three.js 与 OrbitControls 已放入 `vendor/` 文件夹，可离线加载三维库，不依赖在线 CDN、后端、数据库或 API 密钥。

## 主要功能

- 使用 Three.js 构建三维热力环流实验场景。
- 左侧显示红橙色受热地面，右侧显示蓝色冷却地面。
- 受热区上方显示暖色上升气流箭头。
- 冷却区上方显示蓝色下沉气流箭头。
- 受热区近地面显示 `L 低压`，冷却区近地面显示 `H 高压`。
- 近地面气流方向为右侧冷却区 `H` 到左侧受热区 `L`。
- 高空补偿气流方向为左侧受热区上空到右侧冷却区上空。
- 粒子沿闭合路径循环运动，表现完整热力环流。
- 支持五个实验阶段切换，并同步更新右侧讲解文字。
- 支持调节 0℃ 到 20℃ 的地表温差，影响地表颜色、箭头强度、粒子速度和环流明显程度。
- 支持播放、暂停、重置和标签显示隐藏。

## 实验原理

热力环流形成的根本原因是地表冷热不均。受热区近地面空气受热膨胀，密度减小，空气上升；冷却区空气冷却收缩，密度增大，空气下沉。受热区空气上升后，近地面空气减少，形成低压 `L`；冷却区空气下沉后，近地面空气堆积，形成高压 `H`。在近地面，空气由高压区流向低压区，即由冷却区流向受热区；在高空，空气形成方向相反的补偿气流。垂直运动和水平运动共同构成闭合的热力环流。

## 交互说明

- “地表温差”滑块：调节冷热差异强弱。温差越大，热区越红、冷区越蓝，箭头越长越粗，粒子运动越快，环流越明显。
- 阶段按钮：按“地表受热不均、空气垂直运动、气压差异形成、水平气流产生、完整热力环流”的顺序观察形成过程。
- 播放 / 暂停：控制闭合路径上的粒子动画。
- 重置实验：恢复默认温差 10℃、完整热力环流阶段、标签显示、动画播放和默认视角。
- 隐藏 / 显示标签：控制三维场景中的文字标签显示，不影响模型和动画。

## 技术实现

- `index.html`：页面结构、标题区、三维场景容器、说明面板和交互控件。
- `style.css`：现代化课堂展示界面、响应式布局、按钮和标签样式。
- `main.js`：Three.js 场景、相机、灯光、地表模型、箭头模型、闭合曲线、粒子动画、标签投影和交互状态控制。
- Three.js 核心对象包括 `PlaneGeometry`、`BoxGeometry`、`ConeGeometry`、`CylinderGeometry`、`CatmullRomCurve3`、`TubeGeometry` 和 `requestAnimationFrame` 动画循环。
- `vendor/` 文件夹保存普通脚本版 Three.js 与 OrbitControls，本地运行时不依赖在线 CDN，也不依赖 ES Module / import map。

## 教学应用场景

- 高中地理“大气受热过程与热力环流”课堂导入。
- 教师演示热力环流形成的连续过程。
- 学生自主调节温差，探究温差与气流强弱之间的关系。
- 课后复习中对静态教材示意图进行动态补充。

## 科学性说明

本作品严格遵循热力环流的地理原理：受热区空气上升，冷却区空气下沉；受热区近地面为空气减少后的低压 `L`，冷却区近地面为空气堆积后的高压 `H`；近地面空气由高压区流向低压区，高空气流与近地面气流方向相反。作品不使用真实地图、国界、行政区划，也不涉及季风、台风、锋面、气旋、反气旋等无关内容。

## 作品说明

本作品以高中地理“大气受热过程与热力环流”为基础内容，构建了一个基于 AI 辅助开发的 3D 交互式地理实验媒体。作品通过三维场景模拟地表冷热不均条件下空气的上升、下沉和水平运动过程，并使用动态箭头、温度色带、气压标识和分阶段解释，直观呈现热力环流的形成机制。

与传统静态示意图相比，本作品具有较强的过程性和交互性。学生可以通过调节地表温差，观察气流强度和环流结构的变化，从而理解冷热差异、气压差异和空气运动之间的因果关系。该实验媒体适用于高中地理课堂导入、课堂演示、学生自主探究和课后复习等教学场景。
